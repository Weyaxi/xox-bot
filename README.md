# XOX Botu

Python programlama dili ile geliştirilmiş bir XOX (Tic Tac Toe) botudur. Yenmeniz imkansızdır. Bot "O", siz "X" olarak oynarsınız. 

Numaralar 1'den başlar ve 9'a kadar ilerler. 

```
1 | 2 | 3
----------
4 | 5 | 6
---------
7 | 8 | 9
```
